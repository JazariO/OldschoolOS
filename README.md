# Oldschool OS
An oldschool operating system UI built for the Unity Engine.

## Dependencies
- Unity 'New' InputSystem (1.14.2+)
- FMOD for Unity (2.03.10+)
- Proselyte.Sigils (0.1.6-alpha+)
- Unity TextMeshPro Essential Resources

## Installation
You will need to install the following packages before installing this one. Due to limitations with Unity's package distro through git links I can't set these to auto-download for you, so you'll need to install them yourself:
1. FMOD Unity: https://assetstore.unity.com/packages/tools/audio/fmod-for-unity-2-03-311497
2. TextMeshPro Essential Resources: Window > TextMeshPro > Import TMP Essential Resources
3. Proselyte Sigils: https://github.com/JazariO/Sigils.git

Then install the OldschoolOS package: https://github.com/JazariO/OldschoolOS.git

## Features
- Program windows with moveable panels.
- Media Explorer which shows images.
