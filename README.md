# Oldschool OS
An oldschool operating system UI built for the Unity Engine.

## Dependencies
- Unity 'New' InputSystem (1.14.2+)
- FMOD for Unity (2.03.10+)
- Proselyte.Sigils (0.1.6-alpha+)
- Unity TextMeshPro Essential Resources

## Installation
You will need to install FMOD and TextMeshPro Essential Resources before importing this package.

## Features
- Program windows with moveable panels.
- Media Explorer which shows images.

## Planned Features
- Closeable program windows
- Forgrounding program windows
- Changeable cursor
- Taskbar program visibility
- Opening programs via desktop icons
